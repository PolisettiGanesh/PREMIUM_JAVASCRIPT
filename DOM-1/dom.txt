# üå≥ DOM (Document Object Model) ‚Äî Introduction
===================================================

## ‚úî What is the DOM?
    - DOM = A tree-like structure created by the browser.
    - It represents **HTML elements as JavaScript objects**.
    - Allows JavaScript to **access, modify, add, delete** anything in a webpage.

## ‚úî Why Do We Need DOM?
    - JavaScript alone cannot change HTML.
    - DOM gives JavaScript the power to:
    - Change text, colors, styles
    - Add/remove elements
    - Handle user events (click, input, submit)
    - Create dynamic websites


# üåø How Browser Creates the DOM

When Html page will render into the browser,browser will create a tree like structure called DOM.

    HTML ‚Üí Browser reads ‚Üí Produces DOM Tree.

Example:
-----------
<html>
  <body>
    <h1>Hello</h1>
    <p>Welcome</p>
  </body>
</html>

DOM Tree:
------------
- document
  - html
    - body
      - h1
      - p

# üå≥ DOM  Types
-----------------

Node Types
-----------

- Everything in the DOM is a *node*.  
- Each node has a type represented by a number called **nodeType**.
- A generic term for any elements in the DOM tree 

Here are the most important DOM Node Types:

‚úÖ 4. querySelectorAll()

‚úî Selects all matching elements
‚úî Supports any CSS selector
‚úî Returns NodeList (can use forEach!)

Example
<div class="info">A</div>
<div class="info">B</div>
<div class="info">C</div>

<script>
  const boxes = document.querySelectorAll(".info");
  boxes.forEach(box => console.log(box));
</script>

‚≠ê Use Cases of querySelectorAll() (Best Examples)
