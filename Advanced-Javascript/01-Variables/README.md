# 01 - Variables

Introduction to JavaScript variables — declaration, initialization, reassignment, and variable terminology.

## Files

- **01-Variables.md** — Detailed notes on variable concepts
- **index.html** — Browser page to run the JS examples
- **variable-1.js** — Practical examples: `let`, `const`, declaration vs initialization vs reassignment
- **variable-2.js** — Hoisting demo with `var`

## Key Concepts

- `let` — mutable, block-scoped variable declaration
- `const` — immutable binding, must be initialized at declaration
- Declaration vs Initialization vs Reassignment
- Variable terminology (keyword, identifier, type annotation, initial value)
