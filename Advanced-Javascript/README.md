## How to Use

1. Open any `.md` file to read the theory
2. Open the corresponding `.html` files in Chrome
3. Press `F12` to open DevTools → **Sources** tab
4. Follow the step-by-step exercises to see JavaScript internals in action

## Purpose

Built for mastering JavaScript fundamentals through visual debugging — understanding the engine, not just memorizing syntax.
